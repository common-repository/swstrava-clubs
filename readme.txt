Display the recent activities of one of your Strava clubs, on your Wordpress site.

== Description ==

SWStrava Clubs is a widget, allowing you to display the recent activities recorded in Strava, for one of your clubs.

To use the plugin you need:

* A login to Strava.
* To be a member of one or more clubs on Strava.
* Your Strava Access Token.

Within the settings page of SWStrava Clubs you can:

* Select the club you wish to display activities for.
* Decide how names are displayed.
* Select the units used for displaying distances.
* Change the title of the plugin e.g. "Recent activities for my club".
* Choose whether to link each activity to Strava.
* Select the number of activities to display.
* Select your timezone.  

The plugin can be seen in action on this [site](http://www.cwtc.org.uk "SWStrava Clubs")


== Installation ==

To install the plugin:

1. Upload the `swstrava-clubs` directory to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. From the 'Appearance-Widgets' menu, drag the 'SWStrava Clubs' widget to the sidebar of your choice.
1. Access the 'SWStrava Clubs' settings to enter your Strava Access Token and choose the Strava club you wish to display.


The 'swstrava-clubs' directory contains:

* A readme file
* The main plugin file - 'sw_strava.php'
* A stylesheet - sw_strava.css
* Screenshots of the app.
